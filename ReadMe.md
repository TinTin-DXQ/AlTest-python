# 项目介绍：- model.py用于生成模型，保存为model.h5。- generate.py用于生成对抗样本。- attack_data与test_data按照作业要求生成。- 生成attack_data.npy的时间为5647.1349148750305秒# 算法详解：- 将测试集中的图片进行尽可能少的像素点修改，但能使模型识别出更多错误。- 每张图片进行至多三类修改，one_attack对图片中的十对像素点进行交换；two_attack在图片中找出五块像素点进行最大化修改；three_attack作为炸弹，只保留中间的3*3像素点块。如果上一次attack满足模型识别正确则不进入下一次attack。# 个人感受：- 题意的理解和代码的从无到有都是比较困难的。- pycharm跑到三百多个图片就无错跳出了，修改了可用内存大小，改用cpu跑程序之后解决了。- three_attack占比过多，解决方法是增大了前两次修改的像素点个数。- two_attack经过多次修改，从随机五个像素点的修改到随机五块像素点的修改，后来发现随机五块有很大可能性重复就把五块像素点在固定区域随机了。three_attack也由只保留右边和下边缘改成了保留中间部位。# 参考文献：[训练您的第一个神经网络：基本分类Fashion MNIST(tensorflow2.0官方教程翻译)](https://www.jianshu.com/p/d67df190a9f6)  [1.利用TensorFlow进行Fashion MNIST数据集的基本分类问题](https://blog.csdn.net/m0_37393514/article/details/81010587)  [TensorFlow对Fashion MNIST数据集分类](https://blog.csdn.net/ScapeD/article/details/86533772)    [python实现计算SSIM的，并介绍SSIM原理](https://blog.csdn.net/weixin_42096901/article/details/90172534)  [图像质量评估算法 SSIM(结构相似性)](https://blog.csdn.net/chaipp0607/article/details/70158835)
